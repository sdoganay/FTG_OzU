
Here is how to play the replay file from Python.


1. Start FightingICE with argument “--py4j”


2. Execute replay.py 
   
- Write the replay file name (without the extension .dat), of any replay file in \log\replay, that you want to play in replay.py
   
- Write other codes you want to do below "# Main process".

For more information, please read the comments written in replay.py
